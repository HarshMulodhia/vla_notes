## Study flow

The notes now follow a 6-phase, 14-week curriculum aligned with the awesome-vla-study reading flow:
1. generative foundations,
2. early robot foundation models,
3. modern VLA architectures,
4. data scaling,
5. efficient inference and dual-system execution,
6. RL fine-tuning, reasoning, and world models.

Use the structured guide for the weekly sequence and the book for deeper chapter-length notes.
